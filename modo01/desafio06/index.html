Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Parágrafos
    • Use a <p>

  • Títulos
    • Use a tag <h1>

  • Imagens
    • Use a tag <img>

  • Abreviações
    • Use a tag <abbr>

  • Códigos-fonte
    • Use a tag <code>

  • Listas numeradas
    • Use as tags <ol> e <li>

  • Listas com demarcadores
    • Use as tags <ul> e <li>
